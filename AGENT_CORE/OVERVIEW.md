# Project Overview: VNotes
**North Star:** Create an agentic minimalist editor that leverages AI for streamlined note-taking and editing, accessible anywhere as a PWA.
**Core Architecture:** Vite/React (Frontend), Vanilla CSS (Styling), vite-plugin-pwa (PWA support), Stitch (Design/Prototypes).
**Guiding Principles:** Minimalist UI, visual excellence, agentic capabilities, offline availability.
**Constraints:** None currently specified.
